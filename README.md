# 🔥 Flame & Fork | Premium Fine Dining Restaurant Website

Welcome to the official repository for **Flame & Fork**, a premium, elegant, and fully responsive fine dining restaurant website. Designed with rich aesthetics, modern layouts, and premium CSS-only animations and interactions.

---

## ✨ Features

- **Premium Responsive Design:** Seamless layout adjustments across mobile, tablet, and desktop viewports.
- **Pure CSS Interactive Menu:** An elegant tabbed menu filter (Starters, Main Courses, Desserts) powered entirely by CSS radio hacks (no JavaScript required).
- **CSS-Only Navigation:** A fluid, sticky navigation bar with a responsive, CSS-only hamburger menu toggle for mobile viewports.
- **Rich Aesthetic Styling:** Built with a curated dark/gold premium color scheme, custom SVG graphics, smooth hover transitions, and elegant micro-animations.
- **Typography:** Uses Google Fonts pairing (`Playfair Display` for high-end serif headings and `Lato` for readable body text).
- **SEO & Accessibility Ready:** Structured with semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<article>`, `<footer>`), custom ARIA attributes, and unique IDs for interactive components.

## 🚀 Setup and Installation

To view and run this project locally, follow these simple steps:

### Prerequisites
You only need a modern web browser installed.

### Steps to Run
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bunnyvalluri/-Restaurant-website-2026-.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd "-Restaurant-website-2026-"
   ```
3. **Open the site:**
   - Double-click the `index.html` file to open it in your default web browser, OR
   - Run it using a local development server (e.g., Live Server extension in VS Code) for the best local hot-reloading experience.
